# 🚀 GoodDollar Automation & Helper App

## 📖 Description
This project is a web-based GoodDollar helper application designed to automate claiming, manage multiple wallets, and simplify token operations on blockchain networks like CELO and FUSE.

The app also includes an AI chatbot assistant to help users understand features and use the platform easily.

---

## ✨ Features

### 🔗 Batch Claim
- Claim GoodDollar rewards from 100+ wallets at once
- Import private keys or upload CSV files
- Live transaction results and status tracking

### 💰 Batch Token Collection
- Collect tokens from multiple wallets into one master wallet
- Automatic aggregation system
- Real-time success and failure tracking

### 🧮 Balance Checker
- Check balances of multiple wallets simultaneously
- Supports native tokens and G$ tokens
- Export results as CSV

### 👛 Master Wallet System
- Secure destination wallet
- Password protected
- Used for batch collections

### ⏰ Auto Claim Scheduler
- Daily automatic claim setup
- Supports CELO and FUSE networks
- Saves claim schedule in database

### ⛽ CELO Faucet Integration
- Automatically requests gas fees from faucets
- Multiple faucet fallback system

### 🤖 AI Chat Assistant
- Friendly Bengali chatbot
- Explains app features
- Helps users troubleshoot problems

---

## 🛠️ Tech Stack
- Python (HTTP Server)
- SQLite / PostgreSQL Database
- JavaScript + HTML Frontend
- Gemini AI API
- Blockchain RPC (CELO & FUSE)
